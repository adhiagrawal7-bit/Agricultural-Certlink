# CertLink – Agricultural Export Certification System

## Setup Instructions
1. Download the project folder.
2. Open `index.html` in any web browser.
3. Type a certificate ID in the input field.
4. Click **Generate Certificate** to see the certificate preview.
5. Click **Verify Certificate** to verify the certificate ID.

## Tech Stack
- **HTML** – Website structure
- **CSS** – Styling and layout
- **JavaScript** – Interactive buttons and certificate preview

## Features
- Enter a certificate ID (e.g., BAS-2026-01)
- Generate certificate preview instantly
- Verify certificate ID
- Attractive front-end design, easy to use

## Project Overview
CertLink is a digital certification system for agricultural exports. 
It helps farmers generate and verify export certificates instantly, 
reducing paperwork and delays.
